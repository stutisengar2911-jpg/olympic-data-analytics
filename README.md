## 📊 Dataset

The dataset contains historical Olympic Games data including:

- Year and season of Olympic Games  
- Host city and continent  
- Participating countries (NPC)  
- Medal counts (Gold, Silver, Bronze)  
- Sports categories  

This data is used to analyze performance trends, medal distribution, and geographic patterns.

## ▶️ How to Use

1. Download the dataset from the `data/` folder  
2. Open the `.pbix` file in Microsoft Power BI  
3. Explore different dashboard pages  
4. View insights and analysis  

You can also check the presentation for a complete explanation of the project.
